# Student-Management-System


## Overview
This project is a console-based Student Management System developed in C++.  
The system helps in managing student records efficiently using file handling and menu-driven operations.

## Features
- Add new student records
- Display all student information
- Update existing student details
- Delete student records
- Store data permanently using file handling
- Simple menu-driven interface

## Technologies Used
- C++
- Object Oriented Programming (OOP)
- File Handling

## Expected Outcome
The application allows users to manage student information easily with persistent data storage for reliable record management.

## How to Run

### Compile the Program
```bash
g++ main.cpp -o main
```

### Run the Program
```bash
./main
```
